# FIFA-WC-Portal
A static website for the FIFA World Cup 2026 with fixtures, results, stats, live score cards, venues, teams, and official live links.

## Run locally

```bash
python -m http.server 5178
```

Then open `http://127.0.0.1:5178/`.

## Data

The site fetches live match data from ESPN's public FIFA World Cup scoreboard endpoint and links to the official FIFA tournament hub.
